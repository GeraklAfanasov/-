# Flask Text Analysis API

Расчетно-графическое задание по дисциплине «Разработка программных приложений».
Вариант 2. Реализация RESTful API для анализа текста с балансировкой нагрузки (Nginx) и CI/CD (GitHub Actions).

## Функциональность
*   **POST /analyze**: Принимает JSON `{"text": "..."}`, возвращает количество слов и частотный анализ.
*   **Балансировка**: Нагрузка распределяется между портами 5001, 5002, 5003.

## Установка и запуск

1.  **Установка зависимостей:**
    ```bash
    pip install -r requirements.txt
    ```

2.  **Запуск кластера серверов (3 инстанса):**
    ```bash
    python start_instances.py
    ```

3.  **Настройка Nginx:**
    Используйте файл `nginx.conf` для настройки веб-сервера.
